<div align="center">

### [`YSIM`](https://ysim.onrender.com) — *You Sent It. It's Mine.*

**Self-destructing encrypted messaging. No logs. No traces. No regrets.**

[![Live](https://img.shields.io/badge/Live%20App-ysim.onrender.com-20B2AA?style=for-the-badge)](https://ysim.onrender.com)
[![Repo](https://img.shields.io/badge/Source-D--YSIMs%2FYSIM-20B2AA?style=flat-square&logo=github)](https://github.com/D-YSIMs/YSIM)

</div>

<br/>

| Feature | Details |
|---|---|
| 🔒 Encryption | AES-256-GCM end-to-end |
| 💣 Self-destruct | 30-second auto-destruct after read |
| 🚫 Screenshot guard | Canvas-based rendering + print/screenshot blocking |
| 👁️ DevTools trap | Active DevTools detection |
| 🔑 Passphrase auth | Secure hashed passphrase access |
| 🌐 Stack | Node.js · Express · Vanilla JS |
